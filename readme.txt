Jellyfish Fields Directions

1. The objective of the game is to catch all the jellyfish.
2. Click Start on Gameboy or Enter on Keyboard to go to the next part of the game.
2. You can click Select on Gameboy or Backspace on Keyboard at any part of the game to go back to the start screen.
3. Use your arrow keys to move Spongebob directly onto the jellyfish in order to catch them. You can see the amount of jellyfish left on the bottom right of the screen.
4. After catching all the jellyfish, feel free to play again!
